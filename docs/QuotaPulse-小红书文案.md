# QuotaPulse（额度脉搏）
## 给这个 menu bar 小工具起个正式名字 + 一篇小红书种草文案

---

## 一、名字方案

**推荐正式名：QuotaPulse · 额度脉搏**

这个 app 的本质是"常驻 macOS 菜单栏、实时显示 AI API 额度/余额状态的小工具"。原来的工程名叫 GLMQuotaBar，但它其实同时支持 GLM（智谱）和 DeepSeek 两家，所以名字得脱离单一厂商、更通用、更像一个正经产品。

选「额度脉搏」的理由：
- 「额度」直击痛点——所有 AI API 都讲额度/余额，不用解释；
- 「脉搏」传达"实时跳动、抬眼可见"的产品形态，正好对应菜单栏那个随用量变色的小数字，比"管家/监控"更轻盈、更有温度；
- 英文 QuotaPulse 读起来像个正经 indie Mac 应用（参照 iStat Menus、Bartender 这类），上架和出海都站得住。

备选方案：
- QuotaBar / 额度条 —— 最直白，一眼知道是菜单栏条状显示，但偏平淡；
- 额度探针 / QuotaProbe —— 科技感更强，偏开发者向；
- 余额哨兵 / QuotaSentinel —— 强调"站岗守候、到点提醒"，适合后续加重置报警功能时用。

**正文：**

用 AI 写代码的朋友应该都懂——

GLM 的 5 小时窗口、7 天额度，DeepSeek 的账户余额，高峰期还 ×2 计费……这些信息东一个西一个，想看一眼当前还剩多少，得翻好几个网页后台。等你想起来去看的时候，额度往往已经爆了。

所以我做了一个 macOS 菜单栏小工具：**额度脉搏（QuotaPulse）**。

📌 **它能干嘛？**

就一件事：把你的 AI API 额度/余额，实时挂在屏幕顶部的菜单栏里，抬眼就能看到，不用切窗口、不用开网页。

📌 **两个平台，各有妙处：**

**GLM（智谱）**
菜单栏直接显示「5 小时用量% / 7 天用量%」，比如 `32% / 18%`。
按剩余额度变色：剩得多是绿色，快没了变黄，所剩无几直接飘红。点开菜单还能看到精确的重置倒计时（"5.3h 后重置"那种），什么时候额度刷新一目了然。

**DeepSeek**
菜单栏显示账户余额，前面一个小圆点告诉你当前是不是高峰计费时段(官方七月中旬上线)——高峰期（每天 9-12 点、14-18 点，DeepSeek 会 ×2 计费）圆点变红，非高峰变绿。光这一个提醒，就能帮你省下不少高峰期浪费的钱。
菜单里还藏了个彩蛋：一键切换 Flash / Pro 模型，省得去改配置文件。

📌 **为什么放菜单栏？**

因为写代码的时候最不想干的事，就是切到浏览器去查额度。放菜单栏，那个小数字会一直在余光里——写嗨了它默默变黄提醒你"快没额度了该收手了"，比任何通知都好使。

📌 **技术上的小克制**

整个 app 只有一个 Swift 文件、五百多行，编译完就是个轻飘飘的原生应用。纯菜单栏形态，不占 Dock、不弹窗、不后台作妖。每 5 分钟自动拉一次数据，平时不刷接口，不吃电。

📌 **现在怎么用？**

目前它是配合 CC Switch（一个 AI 编程多渠道切换工具）使用的——它会自动读取 CC Switch 里当前选中的渠道和 API key，切到哪个平台就显示哪个平台的额度，无缝衔接。

📌 **后面会更香**

我正在做一个独立版本：到时候不依赖任何工具，直接把 API key 粘进去就能用。无论是 GLM、DeepSeek 还是更多平台，填个 key 就能在菜单栏看额度，谁来都能用。
